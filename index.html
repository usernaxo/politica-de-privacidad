Política de Privacidad

Última Actualización - 20 de Febrero, 2025

Información que recopilamos

SEI Software recopila los siguientes datos cuando el usuario utiliza nuestra aplicación:

  • Correo electrónico empresarial: Utilizado para la identificación del usuario dentro de la aplicación.
  • Clave de acceso: Utilizada para garantizar la seguridad y autenticación del usuario.

Uso de la información

Los datos recopilados se utilizan exclusivamente para los siguientes propósitos:

  • Autenticación segura y acceso a la aplicación.
  • Identificación del usuario dentro de los servicios proporcionados por SEI Software.

Datos mostrados en la aplicación

En la pantalla de la aplicación, se muestran los siguientes datos del usuario:

  • Nombre y apellidos.
  • Correo personal.
  • Dirección.
  • Género.
  • Nacionalidad.
  • Teléfono de contacto.

Compartición de datos

SEI Software no comparte información del usuario con terceros bajo ninguna circunstancia.

Toda la información recopilada a través de nuestra aplicación se utiliza exclusivamente para mejorar la experiencia del usuario y optimizar el funcionamiento de nuestros servicios.

Seguridad de los datos

Implementamos medidas de seguridad para proteger la información del usuario y evitar accesos no autorizados. La clave de acceso es almacenada de forma segura y no es visible para terceros.
